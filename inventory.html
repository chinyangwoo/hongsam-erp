센서 데이터 ERP 연동 안내

향후 시설에 설치된 실제 하드웨어 센서 정보를 이 ERP 시스템으로 자동 연동하기 위한 아키텍처입니다.

  1. 에지 디바이스 (Sensor Edge): 보일러실, 전기실, 스파 등에 설치된 계측기(IoT 장비, ESP32, PLC)가 MQTT 프로토콜 또는 HTTP POST 통신으로 내부 로컬 망 또는 클라우드 API 게이트웨이로 데이터를 송출합니다.
  2. 백엔드 서버 (Backend/DB): 수집 서버에서 초단위 또는 분단위로 전송된 데이터를 정제하여 시계열 속성 데이터베이스(예: PostgreSQL, InfluxDB, AWS Timestream)에 축적합니다.
  3. ERP 프론트 (현재 화면): 브라우저에서 WebSocket이나 SSE(Server-Sent Events) 기술을 사용하여 백엔드와 연결을 유지, 상태가 변동될 때마다 이 화면의 인터랙티브 대시보드가 실시간 업데이트됩니다.

현재는 프로토타입 디자인이 적용되어 있으며, 자바스크립트 시뮬레이터를 통해 3~5초 단위로 수치가 라이브로 변동하도록 구성되어 있습니다.
